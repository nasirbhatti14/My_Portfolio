## 📖 About

This portfolio website is built to present my background as a Software Engineering student at **The Islamia University of Bahawalpur (IUB)**. It highlights my technical skills, hands-on projects, internship experience, and certifications — designed with recruiters and hiring managers in mind.

> **Note:** My primary focus is Software Engineering (Python, Django, backend development, databases) rather than front-end web development. This website's UI was generated using **Google AI Studio** based on a detailed custom prompt, and then reviewed and customized by me with my actual resume content. It serves purely as a presentation layer for my work and skills.

## 🚀 Getting Started

### Prerequisites
Just a web browser — no dependencies needed.

### Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/nasirbhatti14/portfolio-website.git
   ```
2. Navigate to the project folder
   ```bash
   cd portfolio-website
   ```
3. Open `index.html` in your browser
   ```bash
   start index.html   # Windows
   open index.html    # macOS
   ```
